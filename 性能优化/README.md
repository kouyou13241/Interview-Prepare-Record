#### 为什么做优化？
- 项目需要亮点：性能优化在前端非常重要。为了和其他项目区分开，在答辩中想要做出区别点，需要在项目中做相关方面的努力。
- 网站性能确实差：图片太多，首页轮播图4张大图，列表页每一个列表项都有图片，光加载图片就非常慢。我在使用的时候都觉得受不了，更不用说用户。
